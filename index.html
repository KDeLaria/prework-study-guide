HTML

  • The head element contains information about the webpage.
    • Meta element defines data such as character set being used by the webpage.
    • Style elements are where the CSS styles are defined.
    • Base element contains the base URL for the webpage.
    • Link elements connect external rescources to the HTML page.

  • The body element represents the visible content shown to the user.
    • Heading elements are title and subtitles for a section.
    • Paragraph block elements are used to section off a paragraph.
    • List elements are used to create an ordered or unordered list.
    • The image element is used to display an image.
    • The line break element creates an empty line.
    • The anchor element creates a link to a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Border is the edge of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Color is the color that the element will display.
  • Height and width define the size of an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "(message)": commits staged content with a message
  • git checkout branch-name: switches to a branch
  • git pull branch-name: retreive and merge commits from a remote branch
  • git push branch-name: upload local branch to the remote repository branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • If statements are conditional statements that execute code if conditions are true.
  • Arrays are variables that contain more than one item in them.
  • For loops continue executing a section of code until a conditon is reached.
  • A function is a section of code that can be called repeatedly throuought the code. Thus, reducing the the amount of repeated code
